# Contributing to Terminus

Terminus is an internal tool with a small team. We're not actively seeking outside contributions at this time, but we appreciate bug reports and feedback.

## Bug Reports

If you encounter a bug, please open an issue with:

- Steps to reproduce
- Expected vs actual behavior
- macOS version, Terminus version, and whether you're running dev or production build

## Feedback

For feature requests or general feedback, open an issue or reach out directly.
